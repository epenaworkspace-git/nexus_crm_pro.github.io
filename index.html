API Key de Gemini

Alternativa a Anthropic. Se guarda localmente en este dispositivo y tiene los mismos privilegios que la IA de Anthropic dentro de la app.

Déjalo como "gemini-flash-latest" para usar siempre el modelo gratuito más reciente de Google (2.5, 3.0 y futuras versiones se resuelven solas). También puedes escribir un modelo específico, p. ej. "gemini-2.5-flash" o "gemini-3.0-flash".
